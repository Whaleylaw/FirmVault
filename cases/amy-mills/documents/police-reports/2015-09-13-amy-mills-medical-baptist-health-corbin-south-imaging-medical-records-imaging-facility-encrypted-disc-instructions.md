Encrypted Disc Instructions
Encrypted CDs only run on Windows PCs (not on Apple devices). 
1. Insert the disc into computer CD/DVD drive.
2. Select “Run dscstart.exe”.
3. Select “View Images” (this may take a few seconds).
4. Select “I Agree”.
5. Type in the disc password (access code) that you 
received from the Imaging Facility or Hospital.
The password is case sensitive. 
You will get 2 attempts at entering the password. If you 
are rejected because of typos, eject the CD/DVD and 
start again at step #1. 
6. Select an exam from the list and click the 
“Display” button (lighter gray color indicates the exam 
is selected).
7. Top left of toolbar: Left diagram takes you back to 
list of exam(s) / images.
Right diagram takes you to the exam report(s) (if 
available).
8. Select each tab to view the exam results (report).
9. To print a report: Right click, select Print; or, 
select CTRL + P on your keyboard and select Print. 
10. Close Program: Select the X in the upper right 
corner of the application.
