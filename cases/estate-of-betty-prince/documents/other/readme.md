CPHub Records Directory

The Records directory contains all files related to a case that you wish to store.

When a file is added to the Records directory via CPHub, it is also accessbile from the Documents tab on the Casepacer website.  

Files added or removed from CPHub will be reflected on the website.
Documents added or removed from the Casepacer website will be reflected in CPHub.

You may add or remove file and folders to this directory.

NOTE: On Windows computers, the maximum file path is 260 characters. This is a limitation of the operating system, there's nothing we can do about it.

If the CPHub file path is longer than this, that file won't display in Windows File Explorer.

The only solution if this happens is to open the case in your web browser, and to rename the "File Name" (NOT Record Name) so that the full path is shorter than 260.
