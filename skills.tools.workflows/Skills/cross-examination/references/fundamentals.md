# Cross-Examination Fundamentals

## Core principles

### 1. Control the witness
- Leading questions only
- Short, simple questions
- One fact per question
- Never ask "why"

### 2. Know your goals
- What admissions do you need?
- What documents will you use to lock them?
- What's the theory you're serving?

### 3. Use documents as anchors
- Confront with prior statements
- Reference medical records
- Use authoritative texts for experts
- Show inconsistencies — don't describe them

---

## Question structure

### Leading forms
- "You did X, correct?"
- "Isn't it true that…?"
- "You would agree that…?"

### Control techniques
- Loop back to unanswered questions
- Use a document as the anchor when a witness drifts
- Short questions get short answers
- Don't argue — impeach with the document and move on

---

## Preparation checklist

- [ ] Read all witness statements (recorded, written, deposition)
- [ ] Read the relevant documents the witness has touched
- [ ] Write the 3–5 key admissions in plain English
- [ ] Draft a topic-by-topic outline (not chronological)
- [ ] Pre-mark every impeachment exhibit
- [ ] Anticipate likely objections and the response
- [ ] Rehearse the transitions between topics

---

## Outline structure

```
I. TOPIC 1 (→ admission: <plain-language statement>)
   A. Foundation
      Q: <leading>
      Q: <leading>
   B. Admission
      Q: <key question>
   C. Lock it
      Show Exhibit X
      Q: <confirmation>

II. TOPIC 2
   ...
```

---

## Common mistakes to avoid

- Asking open-ended questions
- Arguing with the witness
- Asking one question too many (the rehabilitation opening)
- Not listening to answers
- Losing control of pace
- Showing frustration
- Reading from a script instead of watching the witness

---

## Outputs

- Cross outline (topic-by-topic, admission-focused)
- Pre-marked exhibit set with the purpose of each
- List of key admissions to obtain
- Transitions scripted between topics
