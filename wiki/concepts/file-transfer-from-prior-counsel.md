# File Transfer from Prior Counsel

> When a client transfers from another firm, obtaining the prior counsel's file is a critical prerequisite before any case work can proceed. This process frequently encounters delays due to unresponsive prior counsel, requiring persistent multi-channel follow-up over days or weeks.

## Pattern

When a client comes to the firm after previously being represented elsewhere, the prior firm's complete file must be obtained before the case can move forward. The typical workflow is:

1. Identify the prior firm and case manager contact info
2. Send an initial email requesting the file
3. Follow up by phone if no response within 1-2 business days
4. Escalate (daily calls, multiple contacts at the prior firm)
5. Enlist client/family assistance if prior firm remains unresponsive
6. Eventually receive file and perform cursory review

The entire process can take 1-2 weeks even with persistent follow-up.

## Evidence

- In Case A, the firm called and emailed the prior counsel (a large national firm) daily for over a week with no response. Multiple contacts at the prior firm were tried. The client's family member was eventually asked to assist. The file was received approximately 11 days after the first request.
- Prior counsel may have already sent a demand, taken certain negotiation steps, or received offers that the new firm needs to understand before proceeding.
- In a PL case (Case M), the client insisted on sending the prior attorney's file via postal mail rather than digitally. The firm sent a self-addressed stamped envelope (SASE). Two days later the client was persuaded to try email instead. The SOL was approaching within 60 days, requiring emergency litigation filing on the same day as file setup — before the prior counsel file was fully reviewed. Client cooperation in the transfer method (mail vs email) can itself become a bottleneck.

## Agent Guidance

1. On day 1 of a transfer case, identify the prior firm's case manager by name and direct contact info
2. Send a formal written request (email) for the complete client file on day 1
3. If no response within 48 hours, begin daily phone calls
4. Log every contact attempt in the activity log
5. After 5 business days of no response, ask the client or their authorized representative to contact the prior firm directly
6. Once the file is received, do a cursory review and report key findings to the attorney: existing demand, offers, treatment gaps, outstanding liens, pre-settlement funding
7. Update the case file with all prior-counsel information before moving to negotiation

## Pitfalls

- Assuming the prior firm will respond quickly — large firms often have slow internal processes
- Not documenting contact attempts — this creates gaps in the activity log
- Proceeding with negotiation before understanding what the prior firm already did (e.g., a demand may have already been sent)
- Missing pre-settlement funding obligations that the prior firm arranged
