# Intake and Onboarding Workflow

> The intake process converts a potential client into an active case. It involves collecting personal information, executing the retainer agreement, setting up insurance claims, identifying medical providers, and establishing communication channels. The workflow must be completed rapidly to preserve evidence and begin the claims process.

## Pattern

Standard intake workflow observed across cases:

1. **Initial Contact**: Client (or family member) reaches out, usually by phone. Case manager gathers preliminary accident details, injuries, and insurance information.
2. **Information Request**: Firm requests: Name, DOB, SSN, phone number, copy of health insurance card, names/addresses of hospital and PCP.
3. **Contract Execution**: Intake documents sent via VineSign (electronic signature platform) for completion. Documents include fee agreement, HIPAA authorization, and other intake forms.
4. **Welcome Call**: Once docs are signed, firm conducts a welcome call to confirm treatment providers and introduce the client to the process.
5. **Phase Change**: Case moves from Lead/PNC to File Setup.
6. **Insurance Setup**: Send Letters of Representation to both BI (at-fault) and PIP (client's own) carriers. Request acknowledgment.
7. **Provider Setup**: Add all medical providers to the system. Send records/billing requests.
8. **Phase Change**: Case moves from File Setup to Treatment.

Timeline: This entire workflow typically spans 3-7 days from initial contact.

## Agent Guidance

1. Request all necessary information in the first communication (don't spread it across multiple contacts)
2. Send VineSign intake docs within 24 hours of initial contact
3. If docs are not signed within 48 hours, follow up
4. Conduct the welcome call within 24 hours of signed docs
5. Set up insurance claims (LORs to BI and PIP) within 48 hours of signed docs
6. Identify and add all medical providers within the first week
7. Log the phase change from Lead to File Setup once docs are signed
8. For referral cases (husband calling for wife, etc.), get the actual client to sign the docs

## Pitfalls

- Slow VineSign turnaround — follow up if not signed within 48 hours
- Playing phone tag with the client for days during intake (use text/email as backup)
- Not capturing all insurance information (both BI and PIP carriers) in the initial contact
- Not asking about all medical providers during the welcome call
- Not sending LORs promptly (this delays adjuster acknowledgment and can delay everything downstream)
